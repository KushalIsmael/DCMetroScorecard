Project Overview

Our project recreates Tableau's Makeover Monday challenge from 2016 but utilizes AWS! Makeover Monday is a weekly competition wherein the Tableau Community is challenged to recreate a data visualization using Tableau. For our project, we wanted to recreate this competion utilizing AWS Quicksight to create the visualization and embed the visualization on a public website hosted by an S3 bucket. The data and general guidelines for the Makeover Monday challenge can be found here.
